Файл ClipCreate - для более простой обратоки видео. Скрипт делит длинное видео на клипы по 5 минут, что ускоряет обработку
Файл DataSet - скрипт для создания csv датасета свиней. Бета версия, так как на текущий момент собирается не полный датасет (видит 2-3 свиньи из множества).
Для дальнейшего прогресса необходимо дообучить модель на части размеченых кадров. Скрипт можно запустить в Colab для проверки.
pig_measurements_seg_double - csv файл примера датасета после работы скрипта DataSet
